Requirements

  • Windows 11, Mac OSX, Linux
  • Node.js v12 or higher

Install

Mac (Homebrew)

$ brew install RubriX-Kube-cli

NPM

$ npm install RubriX-Kube-cli

Yarn

$ yarn install RubriX-Kube-cli
